## Using the basic Workspace Starter

How to use this generator locally, essentially for development purposes:

```js
bit new basic my-basic-workspace --load-from /Users/me/path/to/this/dir --aspect my-scope/basic
```

How to use this generator after exporting to a remote scope:

```js
bit new basic my-basic-workspace --aspect my-scope/basic
```
